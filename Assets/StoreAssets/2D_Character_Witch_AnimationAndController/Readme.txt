2D_Character_Witch_AnimationAndController. / Ver 1.0.0
Created by kazuto777

This will be the animation and controller for the "2D_Charactor_Witch" asset.
https://assetstore.unity.com/packages/2d/characters/2d-character-witch-242303

It contains many images that are not included in the animation and controller.
By combining them, you can create a variety of expressions.


This program is not intended to be ideal code, but to be simple and easy to read by anyone.

The sample demo is made on 2021.3.23f
This demo and assets are created assuming 1920x1080px.
